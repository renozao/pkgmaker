Check Environment Variables

Usage

hasEnvar(x)

Arguments

x
environment variable name, as a character vector.

Description

Tells if some environment variable(s) are defined.

Examples

hasEnvar('_R_CHECK_TIMINGS_')



 
TRUE 

hasEnvar('ABCD')
